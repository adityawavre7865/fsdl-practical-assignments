# Assignment 01 — Static Web Page with Text Formatting Tags (Tabular Format)

## Aim
To create a static web page that demonstrates all major HTML text formatting tags, presented in a tabular format.

## Technologies Used
- HTML5

## Description
A single static `index.html` page listing HTML text-formatting elements (bold, italic, underline, subscript, superscript, strikethrough, etc.) organized inside an HTML `<table>`, with each tag demonstrated alongside its rendered output.

## Steps Performed
1. Created the HTML file with a table structure.
2. Added rows demonstrating each text formatting tag.
3. Styled the page with inline CSS for readability.
4. Opened the page in a browser and verified rendering.
