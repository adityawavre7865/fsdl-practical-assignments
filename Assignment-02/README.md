# Assignment 02 — Personal Portfolio Page

## Aim
To design a personal portfolio page using HTML and CSS (Bootstrap CDN used for layout/styling).

## Technologies Used
- HTML5
- CSS3
- Bootstrap (CDN)

## Description
A single-page personal portfolio (`portfolio.html`) featuring an introduction/hero section, about section, skills, and other standard portfolio sections, built with Bootstrap components for responsive layout.

## Steps Performed
1. Planned the portfolio sections (About, Skills, Projects, Contact).
2. Built the layout using Bootstrap's grid and components via CDN.
3. Styled the page for a clean, responsive look.
4. Tested responsiveness across screen sizes.
