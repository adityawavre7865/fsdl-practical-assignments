# Assignment 05 — JourneyHub (Travel Agency Website)

## Aim
To create a simple functional travel agency website using Node.js and Express with a MongoDB backend.

## Technologies Used
- Node.js, Express.js
- MongoDB (mongodb driver)
- HTML, CSS, JavaScript

## Description
JourneyHub lets users browse curated travel packages, book a trip with a generated booking reference number, look up an existing booking, and send enquiries via a contact form. Full details, features, and run instructions are in `source-code/README.md`.

## Steps Performed
1. Designed the package browsing, booking, and contact flows.
2. Built the Express server and MongoDB data models.
3. Implemented booking reference generation and lookup.
4. Built a responsive frontend (HTML/CSS/JS) served from `public/`.
5. Tested booking creation, lookup, and contact form end-to-end.
