# Assignment 07 — Student Feedback System

## Aim
To design a web application for a student feedback review system with a MongoDB backend, using React.js and Express.

## Technologies Used
- React (frontend), Node.js, Express.js (backend)
- MongoDB
- JWT authentication

## Description
A role-based feedback system (Student, Faculty, Admin) with JWT authentication, where students submit feedback, faculty view it, and admins manage/delete entries. Full details and run instructions are in `source-code/README.md`.

## Steps Performed
1. Designed role-based access for Student, Faculty, and Admin.
2. Built JWT-based authentication on the Express backend.
3. Built REST APIs for submitting, viewing, and managing feedback.
4. Built the React frontend with dashboards per role.
5. Tested login, submission, and role-based views end-to-end.
