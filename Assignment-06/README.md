# Assignment 06 — Doctor Appointment Booking System

## Aim
To create an online appointment booking application using Node.js and Express with a MongoDB backend.

## Technologies Used
- Node.js, Express.js
- MongoDB
- HTML, CSS, JavaScript

## Description
A doctor appointment booking system where users can view doctors, book appointments, check slot availability, and cancel bookings, with slot-conflict detection on the backend. Full details and run instructions are in `source-code/README.md`.

## Steps Performed
1. Designed the doctor listing and appointment booking flow.
2. Built REST APIs with Express for booking, availability, and cancellation.
3. Implemented slot conflict detection in MongoDB.
4. Built a responsive frontend and confirmation flow.
5. Tested booking, conflict handling, and cancellation.
