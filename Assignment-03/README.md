# Assignment 03 — Dynamic Fashion Store / E-Commerce Web App

## Aim
To design a dynamic web app for a fashion store / e-commerce product listing using HTML, CSS, and JavaScript (Bootstrap CDN used where required).

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap (CDN)

## Description
A multi-page e-commerce-style site with the following pages:
- `index.html` — landing page
- `shop.html` — product listing / shop page
- `collections.html` — product collections
- `about.html` — about the store
- `contact.html` — contact form
- `style.css`, `script.js` — shared styling and dynamic behavior (e.g. filtering, cart-style interactions)

## Steps Performed
1. Designed the multi-page site structure and navigation.
2. Built product listing and collection pages.
3. Added JavaScript for dynamic/interactive behavior.
4. Styled all pages for a consistent storefront look.
5. Verified navigation and functionality across pages.
