# Assignment 04 — Interactive Data Visualization Dashboard

## Aim
To create an interactive data visualization dashboard for a real-life application (weather dashboard), visualizing data using Chart.js.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Chart.js (CDN)

## Description
A dashboard (`index.html`, `style.css`, `script.js`) that visualizes weather-related data using interactive charts built with Chart.js, with a responsive dashboard layout.

## Steps Performed
1. Designed the dashboard layout with card/panel sections.
2. Integrated Chart.js and configured chart types for the dataset.
3. Added interactivity (dynamic updates/filters) via JavaScript.
4. Styled the dashboard for readability and responsiveness.
5. Tested the dashboard in the browser.
